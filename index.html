Hi

Hi — a tiny, symbolic, AI-first programming language

Keep the core mostly symbols so everyone can read/write it. Let humans and AIs meet at the shortest path.

Shape of a program
In Hi, a Block is both an object and a function. The last expression returns. Private fields start with #.
Made with markdown-it, highlight.js (JS mode), and lucide icons.
Hi is early and evolving. Syntax and semantics may change.
